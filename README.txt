    This is a program that takes in a string containing a-z, 0-9, and !/?, and prints the string
using astericks (*) to form the characters. The output can be modified in the AStarDriver.java 
using functions created in AStar.java.
    Ensure all files are together and your system compiles/runs Java

to compile: javac AStarDriver.java
to run:     java AStarDriver
